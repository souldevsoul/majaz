### File Formats
- **Web Display:** WebP (lossy) for modern browsers, JPEG fallback
- **Hero Images:** WebP at 85% quality or AVIF for best compression
- **Thumbnails:** WebP at 75% quality
- **Print Assets:** PNG or TIFF at 300 DPI

### Color Profiles
- **Web:** sRGB IEC61966-2.1
- **Print:** Adobe RGB (1998) or ProPhoto RGB

### Node.js Generation Script
```javascript
// scripts/generate-brand-images.js
import Replicate from 'replicate';
import fs from 'fs';

const replicate = new Replicate({
  auth: process.env.REPLICATE_API_TOKEN,
});

const PROMPTS = {
  majaz: {
    hero: "Ultra-luxury SUV parked on polished black marble forecourt...",
    // ... (prompts from above)
  },
  avtocert: {
    hero: "Modern sedan on clean white background...",
    // ...
  },
  // ... other projects
};

async function generateImage(project, imageType, prompt) {
  console.log(`Generating ${project} ${imageType}...`);

  const output = await replicate.run(
    "black-forest-labs/flux-1.1-pro",
    {
      input: {
        prompt: prompt,
        aspect_ratio: "16:9",
        output_format: "webp",
        output_quality: 90,
        safety_tolerance: 2,
      }
    }
  );

  // Download and save
  const response = await fetch(output);
  const buffer = await response.arrayBuffer();
  const filename = `public/images/${project}/${imageType}.webp`;

  fs.mkdirSync(`public/images/${project}`, { recursive: true });
  fs.writeFileSync(filename, Buffer.from(buffer));

  console.log(`âœ“ Saved: ${filename}`);
}

// Generate all images
async function generateAll() {
  for (const [project, images] of Object.entries(PROMPTS)) {
    for (const [imageType, prompt] of Object.entries(images)) {
      await generateImage(project, imageType, prompt);
      await new Promise(resolve => setTimeout(resolve, 2000)); // Rate limit
    }
  }
}

generateAll().catch(console.error);
```

### Run the Script
```bash
node scripts/generate-brand-images.js
```

## ðŸ’¡ Pro Tips

1. **Consistency:** Use same camera/lens specs across images for same project
2. **Lighting:** Match lighting mood to brand identity
3. **Color Grading:** Apply brand colors in post-processing
4. **Composition:** Follow rule of thirds, leading lines
5. **Sharpness:** Slightly over-sharpen for web display
6. **Compression:** Use modern formats (WebP, AVIF) for better quality at smaller file sizes
